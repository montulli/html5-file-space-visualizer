Where Is My Disk Space?
       an HTML 5 based File system visualizer
Sponsored by:

Click HERE to start the visualizer

This tool allows you to visualize the directories and space used on your computer with the power of modern Web Browsers and HTML 5.
You can use this to quickly find out which directories contain the most data.

Note: This site requires a modern browser with SVG support and a fast JavaScript engine. FireFox 10+, Chrome 10+, etc...
